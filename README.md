# Netherlands Travel Plan Site

This repository hosts a generated static HTML page for sharing a travel plan with reviewers.

Published site: https://personaltravelplan.github.io/Netherlands-site/

## How Updates Work

The HTML and assets in this repository are generated. Do not edit generated site files here directly. Site updates should arrive as pull requests, and GitHub Pages refreshes after the pull request is merged into `main`.

## Privacy Note

This published site is public. Anyone with the link can access the generated page and assets. The generated site includes `robots.txt` and `noindex,nofollow` metadata to reduce search indexing, but those are not access controls.

Do not publish passport details, ticket numbers, hotel confirmation numbers, personal IDs, private addresses, or other sensitive travel documents here.
